# Skill: close-ticket

Usa esta skill al final de una tarea para verificar si realmente está lista para cerrarse.

## Checklist de cierre

1. objetivo cumplido;
2. alcance respetado;
3. líneas rojas no violadas;
4. validaciones ejecutadas;
5. prueba manual mínima definida o ejecutada;
6. documentación actualizada;
7. mensaje de commit sugerido.

## Formato de salida

- estado: cerrado | parcialmente cerrado | no cerrar
- resumen
- archivos tocados
- validaciones
- deuda o riesgo residual
- commit sugerido
