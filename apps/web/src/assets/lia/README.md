# Assets de Lía Andina — integración v1

Imágenes PNG con fondo transparente de Lía Andina para uso en la app.
Derivadas del turnaround visual cerrado en `docs/avatars/lia-andina/`.

## Estados disponibles

| Archivo | Fuente original | Uso en app |
|---|---|---|
| `lia-idle.png` | `turnaround/a1-front.png` | Layout header (estado base) |
| `lia-three-quarter.png` | `turnaround/a2-three-quarter-front.png` | Usos secundarios con ángulo |
| `lia-calm.png` | `details/b3-state-calm.png` | FinalScreen (cierre contemplativo) |
| `lia-curious.png` | `details/b4-state-curious.png` | IntroScreen (inicio del recorrido) |

## Notas

- Estas imágenes son placeholders de alta calidad hasta que se produzca el modelo 3D de Lía.
- Cuando se integre el pipeline 3D (Meshy/Tripo → rig → animaciones), estos PNG se reemplazarán por renders o animaciones en tiempo real.
- No editar manualmente. Si hay que regenerar, hacerlo desde la biblia visual en `docs/avatars/lia-andina/`.
