# Lía Andina — concept art de producción

Esta carpeta contiene concept art de producción para Lía Andina.

Se usa para selección visual, revisión y aprobación antes de Blender.

## Estructura

- `candidates/`: propuestas sin aprobar.
- `approved/`: piezas aprobadas para avanzar.
- `rejected/`: piezas descartadas con motivo.

## Reglas

- No es runtime.
- No importar desde React.
- No sustituir los PNG fallback.
- No crear multiavatar.
- No contradecir ADR-001.
