# Producción 3D — Lía Andina

Esta carpeta reúne referencias, prompts, scripts, fuentes Blender, exports y validaciones para producir Lía Andina como guía 3D.

Lía Andina es la guía única del proyecto.

No crear multiavatar.
No cambiar ADR-001.
No usar boca, nariz, cejas, brazos, piernas ni alas de hada.
No convertir a Lía en personaje humanoide.

## Subcarpetas

- `00-source-docs/`: referencias documentales.
- `01-fallback-png/`: copias de referencia de los PNG runtime actuales.
- `02-reference-sheets/`: model sheets y hojas visuales.
- `03-concept-art/`: candidatos/aprobados/rechazados.
- `04-blender-prompts/`: prompts para scripts Blender.
- `05-blender-scripts/`: scripts Python generados/aprobados.
- `06-blender-source/`: archivos `.blend`.
- `07-exports/`: GLB candidatos y aprobados.
- `08-previews/`: renders de revisión.
- `09-validation-notes/`: notas de peso, performance y aprobación.
