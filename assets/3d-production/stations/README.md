# Producción 3D — Estaciones

Esta carpeta organiza la producción 3D por estación.

Cada estación debe pasar por:

1. definición UX;
2. concept art;
3. aprobación visual;
4. handoff a Blender;
5. script Blender;
6. fuente `.blend`;
7. export GLB;
8. validación;
9. integración React posterior.

No integrar assets desde esta carpeta en runtime.
